# Knit UI

**A cross-platform (React Native + Web) component kit and design system.**

This site aggregates the Storybook of every published `@knitui/*` package. Pick a
package in the sidebar (each is a composed Storybook) to browse its stories:

| Package | What's inside |
| ------- | ------------- |
| **Components** | The core component kit — ~100 cross-platform components. |
| **Dates** | Calendars, date/time pickers, and inputs. |
| **Carousel** | Cross-platform carousel / slider. |
| **Graphics** | Skia-powered graphics, effects, and audio visualizers. |
| **Icons** | The Tabler-derived icon kit. |
| **Map** | MapLibre map surface. |
| **Media** | Audio & video players, playlists, and visualizers. |
| **Sheet** | Cross-platform bottom sheet. |

`@knitui/core`, `@knitui/hooks`, and `@knitui/plugins` have no stories of their
own — they power everything above.

Everything renders on the **web** target via `react-native-web`; the same
components run natively on iOS/Android from the same source.
